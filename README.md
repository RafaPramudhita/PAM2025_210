# RentalinAja 🚗

**RentalinAja** adalah aplikasi penyewaan (rental) kendaraan berbasis Android yang dirancang untuk memudahkan pengguna dalam memproses pemesanan kendaraan dan membantu admin dalam mengelola inventaris serta riwayat transaksi.

## 🚀 Fitur Utama

- **User Side:**
  - Login & Register (Autentikasi).
  - Home Screen: Menampilkan daftar kendaraan yang tersedia.
  - Detail Kendaraan: Informasi lengkap mengenai spesifikasi dan harga.
  - Booking: Proses pemesanan kendaraan secara real-time.
  - Riwayat: Melacak status pemesanan (Pending, Aktif, Selesai).

- **Admin Side:**
  - Dashboard Admin: Ringkasan statistik rental.
  - Manajemen Kendaraan: Tambah, edit, dan hapus data armada.
  - Validasi Booking: Menyetujui atau membatalkan pesanan masuk.

## 🛠️ Stack Teknologi

- **Bahasa Pemrograman:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose) (Modern Android Toolkit)
- **Architecture:** MVVM (Model-View-ViewModel)
- **DI Framework:** Hilt / Koin (opsional, sesuaikan dengan file Anda)
- **Database/API:** Firebase / Retrofit (sesuaikan dengan implementasi Anda)
- **Asynchronous:** Kotlin Coroutines & Flow
